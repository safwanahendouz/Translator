# 🌐 Python Translator App (Tkinter)

A simple desktop translation application built using **Python** and **Tkinter**. This app allows users to input text, select source and target languages, and instantly get translations using an external translation API.

---

## ✨ Features

- 🖥️ Clean and simple GUI with Tkinter  
- 🌍 Supports multiple languages  
- 🔽 Dropdown menus for language selection  
- 🔁 Real-time translation on button click  
- ❌ Clear button to reset fields  
- ⚠️ Basic error handling
